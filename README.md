## 项目使用技术：

1. 前端：React hooks + Redux + React-router-dom + axios
2. # 后端：json-server
3. 组件库：antd
4. 后端：json-server

## 项目启动

1. 启动后端

```
npm run json-server
```

2. 下载依赖

```
npm install
```

3. 启动项目

```
npm start
```
